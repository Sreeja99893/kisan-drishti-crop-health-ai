**Kisan Drishti – Smart Crop Disease Detection
Overview**

  Kisan Drishti is an AI-powered application that helps farmers identify crop diseases by uploading photos of affected plants. The system provides quick diagnosis, treatment suggestions, and prevention tips in local Indian languages.

**Problem**

  Many farmers in rural India do not have quick access to agricultural experts. This leads to delayed treatment, overuse of pesticides, and heavy crop losses.

**Solution**

  Kisan Drishti uses image recognition and cloud technology to:

   Detect crop diseases instantly

   Suggest correct treatments and dosage

   Support voice and regional languages

   Work in low or poor internet conditions

**Key Features**

   AI-based crop disease detection

   Support for Hindi, Tamil, Telugu, and Punjabi

   Offline-friendly design

   Accurate pesticide and prevention advice

   Secure and fast cloud processing

**AWS Services Used**

  AWS Amplify – App hosting and authentication

  Amazon S3 – Image storage

  AWS Lambda – Backend processing

  Amazon SageMaker – AI/ML model hosting

  Amazon DynamoDB – Disease and treatment data

  Amazon Translate – Language translation

  Amazon API Gateway – Secure API management

**Impact**

   Reduces crop loss

   Lowers unnecessary pesticide use

   Helps farmers take faster action

   Supports sustainable agriculture in India

**Project Status**

  This project is a hackathon prototype designed to demonstrate how AI and AWS can improve agricultural outcomes for small and marginal farmers.
